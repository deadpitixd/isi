Input Output

Input and Output is more similar to python, but it's still simple.

Here, we want to get a string from user input, and output it.

Example:

string x;
x = input();
cout(x);

Input: Hello World

Output: Hello World

First, you define the variable. Then you set the variable to
input();
And then, the user can input anything into the variable.
After this, the function
cout(any);
outputs the data, or variable into the console.

cout() Accepts many inputs in one function, separated by commas.

Example:

cout(x, 10, "string", 436.412);

When you use two cout's, you might see that they print on the same line.
To fix this, you also need to pass in 'endl'

It's like using '\n' in C++.
