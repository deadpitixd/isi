cout()

Allows user to output text into the command line.

Syntax:

cout(arguments [any]);

cout() can take as many arguments as it is required, but they must have commas in-beetween them.
There also exist ANSI codes built into accessible by calling these variables:

Variable Type Effect
endlUtilityNew Line
col_resetControlReset Styles
col_clearControlClear Screen
col_boldStyleBold Text
col_underlineStyleUnderline Text
col_redColorStandard Red
col_greenColorStandard Green
col_yellowColorStandard Yellow
col_blueColorStandard Blue
col_cyanColorStandard Cyan
col_whiteColorStandard White
col_b_redColorBright Red
col_b_greenColorBright Green
col_b_cyanColorBright Cyan
col_bg_redBackgroundRed Background
col_bg_blueBackgroundBlue Background
