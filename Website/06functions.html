Functions

I personally think that functions are the most complex thing in ISI.
There are two types of functions:
- ISI Functions
- External Functions (C++)

The easier ones are ISI functions as they are similar to C++.

Syntax:

int myFunc(int x){ return x; }

Here, the program defines a function named 'myFunc()' with arguments.
The argument (int x) can only be accessed by the function.

To use it later, you just use myFunc() with an argument inside.
For Example:

cout(myFunc(5));
