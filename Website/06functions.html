Functions

Native ISI functions were added in ISI [[isiVersionCommit:A2]]
External C++ functions were added in ISI [[isiVersionCommit:A4]]
Functions are the most important part of any programming language,
as they allow for repeating code in a clean way.

Creating functions is extremely similar to C++,
first you specify the data type, then you name the function,
and you add arguments separated by commas.
After that, you put code that the function executes in curly brackets.

Example: 

int myFunction(int argument1, float argument2, string argument3){
    print(argument1, argument2, argument3, "\n");
    return 0;
}

To call a function, you do this: 
myFunction(1, 3.141, "text");
Functions introduce a new keyword, 'return'.
Return simply returns a value int non-void functions.
[[code-begin]]int add(int a, int b){ return a + b; } print(add(2,4),"\n"); [[code-end]] This code prints 6, because print() takes the returned value of add()
as an argument.

C++ Functions

C++ Functions work as extensions to the language.

To write and extension, you start the C++ file like this: [[code-begin]]#include "other.hpp" other.hpp is in std/ folder in ISI source code, it has all neccesary includes. [[code-end]] Then, you write all functions in 'extern "C"{}'
All functions must have 'std::vector< Value > args' in them to work correctly.
Example: [[code-begin]]#include "other.hpp" using std::string; extern "C" { Value input(const std::vector& args) { string out; std::cin >> out; return out; } } [[code-end]] To use these functions in isi, you use the extern() function: [[code-begin]]lib x = loadLibrary("library name"); extern(x) string 'function name in C++'(); [[code-end]] [[docVersion]] [[isiVersionCommit:A4]]
