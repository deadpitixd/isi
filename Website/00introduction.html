Interscript Introduction

Welcome to ISI (Interscript Interpreted), here you'll learn how to code
This langauge is simple, almost python-like simple, but has things similar to C++.

First, to run the code, use:

./isi filename.isi;

Most functions (Blue Keywords) can be clicked, and they will lead to their reference page.
Try to hover over input() and click it.

For example:

input();
