Variables

Variables in ISI are easy to understand, they are similar to C++.
Currently there are 4 data types, and they are:

int - Contains a whole number (1, -5, 64)
float - Contains a floating point number (0.0, 63.5, -14.325)
string - Contains a string of characters ("Hello World!")
bool - Stores true or false

Variable definition:

int x;

The interpreter creates a variable named 'x'.

It's default value is 0,

Here, a variable with a value is initialized:

int x = 10;

Variable 'x' is defined with the value 10

The most important part of variables is math, to use math you use the 4 basic operators:

+
-
*
/

Their usage looks like this:

x = 10 + 3;

Or with variables:

x = 10 + y;
